Cómo decide la categoría — Cascada

CAPA 0

Reglas tuyas

Memoria y reglas personales por usuario. Aprende de correcciones.

 CAPA 1

Reglas globales

Patrones regex/literal/contiene curados manualmente.

 CAPA 2

MCC

Código MCC del input o inferido por nombre de comercio.

 CAPA 3

IA (Ollama)

Fallback async cuando ninguna capa anterior matchea.
